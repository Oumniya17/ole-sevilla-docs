# Olé Sevilla

> Where Andalusian culture meets artificial intelligence.

Olé Sevilla es una plataforma web inmersiva diseñada para transformar la manera en la que turistas y locales descubren la ciudad de Sevilla.

Más que una guía turística tradicional, el proyecto propone una experiencia interactiva, emocional y tecnológica donde la historia, la música y la cultura sevillana cobran vida mediante inteligencia artificial, gamificación y diseño visual moderno.

---

# 🎯 Objetivos del Proyecto

- Crear una experiencia turística moderna e interactiva.
- Aplicar inteligencia artificial al turismo cultural.
- Diseñar una interfaz visual inspirada en la estética andaluza contemporánea.
- Fomentar la interacción entre turistas y habitantes locales.
- Crear una plataforma escalable preparada para futuras ciudades.

---

# 🌍 Filosofía

Olé Sevilla combina tradición y tecnología para crear una experiencia cultural inmersiva donde cada rincón de Sevilla puede hablar, cantar y contar su historia al usuario.

---

# 🚀 Tecnologías Principales

- React + Vite
- Node.js + Express
- MongoDB
- TensorFlow.js
- Leaflet + OpenStreetMap
- Framer Motion
- APIs de traducción y reconocimiento musical
